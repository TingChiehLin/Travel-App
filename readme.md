## Getting Started

Step 1. Download or Clone the project:
```
git clone XXX [folder_name]
```

Step 2. Install third party package
```
npm install
```

Step 3. Start server
```
npm start
```

Step 4. Setup the development environement
```
npm run build-dev
```

Step 5. Setup the production environement
```
npm run build-prod
```

Step 6. Testing
```
npm run test
```
